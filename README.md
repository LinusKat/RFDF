# RFDF
I am still not happy with some of the code and structure in RFDF however I have made it public so friends can use it.

RFDF has been battle tested in Project Flight with 60+ million visits and works as intended so feel free to use it. I personally like these types of frameworks more than others, and I decided to make my own as a replacement for the module loader hell we had in Project Flights recode for an entire year.

## Installation
```bash
git clone https://github.com/example/RFDF.git
cd RFDF
```
Place `RFDF` in replicated storage, `init.client.luau` and `init.server.luau` into appropriate directories.
