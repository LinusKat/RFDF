## What is RFDF?

RFDF is a strictly typed framework for organizing networking and services for use in roblox game development. It has own networking sysem for: 
ReplicatedRegistries/RemoteFunctions/RemoteEvents, using endpoints. It also features built in debugging, and the ability for custom logging.

## Why should I use RFDF?

RFDF has been battle tested in Project Flight with 60+ million visits, and works as intended. The inspiration to build it 
came from the decision to make a replacement for the module loader hell we had in Project Flight's recode for an entire year.
